准
第一步：准备工作

系统要求确认

Windows 10 / 11 · 64位
  • 操作系统：Windows 10 或 11（64位）
  • 内存：至少 4GB RAM（推荐 8GB）
  • 硬盘：至少 2GB 可用空间
  • 分辨率：1366×768 或更高
如何查看： 按 Win+R → 输入 msinfo32 → 回车
查看「系统类型」（需为 64 位）和「已安装的物理内存」
Mac Ventura 13+ · Intel / M 芯片
  • 操作系统：macOS 13 Ventura 或更高
  • 内存：至少 4GB RAM（推荐 8GB）
  • 硬盘：至少 2GB 可用空间
  • 芯片：支持 Apple M1/M2/M3 和 Intel
如何查看： 点击左上角苹果图标 →「关于本机」
查看「芯片」和「内存」信息
03 / 14
注
第二步
02

注册 Tableau Public 账号
必须先注册才能下载

1
打开官网 访问 public.tableau.com/app/discover
2
点击注册 右上角 "Sign In" → "Create one for free"
3
填写信息 名字、姓氏、邮箱、密码（至少 8 位，含字母和数字）
4
邮箱验证 去邮箱查收验证邮件，点击链接激活账号
没收到验证邮件？ 检查垃圾邮件文件夹 · 确认邮箱地址正确 · 等 5 分钟后点"Resend Email"

载
第三步
03

下载 Tableau Public

1
登录后进入下载页 点击右上角头像 → "Download Tableau Desktop Public Edition"
2
系统自动识别版本 Windows 用户下载 .exe 文件 · Mac 用户下载 .dmg 文件
3
耐心等待下载完成 文件约 400–600 MB，请确保网络稳定
下载很慢？ 可以尝试挂 VPN 重新下载，官网服务器在海外

第四步 · Windows

Windows 安装步骤

运行安装程序 Step 1
1
找到下载的 .exe（通常在"下载"文件夹）
2
双击运行，弹出「用户账户控制」点击 "是"
3
点击 "Install"，等待进度条完成（约 3–8 分钟）
4
安装完成后点击 "Finish"
首次启动登录 Step 2
1
桌面会出现 Tableau Public 图标，双击打开
2
首次启动要求登录账号
3
输入注册时的邮箱和密码，登录成功即安装完成 ✓

第四步 · Mac

Mac 安装步骤

安装应用 Step 1–2
1
找到下载的 .dmg 文件，双击挂载
2
将 Tableau Public 图标拖入右侧 Applications 文件夹
3
等待复制完成
首次启动 Step 3 · 重要
1
打开「应用程序」，找到 Tableau Public
2
不要直接双击！ 右键点击 → 选择「打开」
3
弹出警告对话框，点击「打开」确认
4
输入邮箱和密码登录，安装完成 ✓
为什么要右键打开？ Mac 安全机制对未经 App Store 下载的软件有限制，第一次需右键→打开，之后正常双击即可

错
常见问题 · Windows

Windows 报错解决方案（一）

缺少 .NET Framework
控制面板 → 程序 → 启用或关闭 Windows 功能
勾选 .NET 3.5 和 .NET 4.8，安装后重试
打不开，提示 0xc000007b
缺少 Visual C++ 运行库
官网下载 Microsoft Visual C++ Redistributable（x64 版），安装后重启电脑
安装进度条卡住不动
1. 临时关闭杀毒软件
2. 右键安装程序 → 以管理员身份运行
电脑是 32 位系统，无法安装
Tableau 不支持 32 位 Windows，无法强行安装
临时方案：使用 Tableau Public 网页版（功能受限）
09 / 14
乱
常见问题 · Windows

Windows 报错解决方案（二）

Q
安装后中文显示乱码 右键桌面图标 → 属性 → 兼容性 → 勾选「以管理员身份运行」
控制面板 → 区域 → 管理 → 系统区域设置 → 勾选「Beta：使用 Unicode UTF-8」
Q
软件一直停留在「加载中」界面 检查网络连接（首次启动需联网）· 关闭防火墙/杀毒软件后重试 · 完全关闭等 30 秒后重开
Q
安装后无法保存文件 Tableau Public 不能保存到本地，必须发布到云端
服务器菜单 → 发布到 Tableau Public → 登录后给工作簿命名并保存

Mac
常见问题 · Mac

Mac 报错解决方案（一）

提示「已损坏，无法打开」
系统设置 → 隐私与安全性 → 找到 Tableau 的提示 → 点「仍然打开」
如仍不行，终端输入：sudo xattr -cr /Applications/Tableau\ Public.app
M1/M2/M3 芯片运行缓慢或崩溃
确认下载了最新版本
旧版本可右键 → 显示简介 → 勾选 「使用 Rosetta 打开」
界面显示英文，如何改中文
苹果菜单 → 系统设置 → 通用 → 语言与地区
确认「首选语言」里简体中文排在第一位，重启 Tableau
地图无法显示，只有空白底图
网络问题，Tableau 地图需访问海外服务器
建议使用 VPN 或切换至手机热点，课程练习可暂时跳过地图部分

缓
常见问题 · Mac

Mac 报错解决方案（二）

Q
Tableau 崩溃，无法正常打开数据 完全退出（Cmd+Q）→ 访达按 Cmd+Shift+G 输入 ~/Library/Application Support/Tableau
找到 Caches 文件夹，删除里面的内容（不要删文件夹本身）→ 重新打开 Tableau
Q
通用：登录时提示「账号或密码错误」 确认邮箱地址无误（注意无多余空格） · 点"Forgot Password"重置密码 · 确认邮箱验证已完成
Q
通用：软件一直停留在「加载中」 检查网络（首次启动需联网） · 关闭防火墙/杀毒软件后重试 · 等 30 秒后重开
